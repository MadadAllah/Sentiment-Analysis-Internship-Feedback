🎯 Project Title: Sentiment Analysis on Internship Feedback¶

🔍 Objective¶

To evaluate the emotional tone of intern feedback through sentiment analysis, aiming to identify strengths and improvement areas for enhancing the internship experience

🛠 Tools & Libraries¶

  • Python: Core programming language
  • Pandas: Data manipulation and analysis
  • NLTK: Natural Language Toolkit (used for sentiment analysis)
  • VADER (SentimentIntensityAnalyzer): For classifying sentiment
  • Matplotlib: Data visualization
  • WordCloud: For visualizing frequent words by entiment

📁 Dataset Description¶

A synthetic dataset containing over 1,000 feedback responses, designed to simulate real-world internship feedback. The feedback includes a mix of positive, neutral, and negative sentiments to mimic realistic intern responses.con')

✍️ Sample Feedback¶

Here are a few randomly selected feedback entries:

  • “I really enjoyed the team sessions.” → Positive
  • “The tasks were unclear and confusing.” → Negative

🔢 Sentiment Scoring Logic¶

Using VADER's compound score:

  • Compound ≥ 0.05 → Positive
  • Compound ≤ -0.05 → Negative
  • Otherwise → Neutral

Apply Sentiment Scoring¶

In [33]:
sia = SentimentIntensityAnalyzer()
df['Sentiment_Score'] = df['Feedback'].apply(lambda x: sia.polarity_scores(x)['compound'])

def classify_sentiment(score):
    if score >= 0.05:
        return 'Positive'
    elif score <= -0.05:
        return 'Negative'
    else:
        return 'Neutral'

df['Sentiment'] = df['Sentiment_Score'].apply(classify_sentiment)

Explanation:

  • Creates a SentimentIntensityAnalyzer object to perform sentiment analysis.
  • Applies VADER to each row in the 'Feedback' column of the dataset.
  • Extracts the compound score from the sentiment analysis result, which is a single number between -1 (most negative) and +1 (most positive).
  • Saves the computed sentiment score in a new column called 'Sentiment_Score' for later analysis.

# Define themes and related keywords
themes = {
    'Mentorship': ['mentor', 'mentorship', 'guide'],
    'Communication': ['communication', 'reply', 'response'],
    'Projects': ['project', 'task', 'assignment'],
    'Learning': ['learn', 'knowledge', 'skills'],
    'Support': ['help', 'support', 'assist']
}

# Function to check sentiment score for each theme
def analyze_sentiment_by_theme(df, themes):
    for theme, keywords in themes.items():
        df[theme] = df['Feedback'].apply(lambda x: any(keyword in x.lower() for keyword in keywords))
        sentiment_score = df[df[theme]]['Sentiment_Score'].mean()  # Average sentiment for the theme
        print(f"{theme} - Avg Sentiment Score: {sentiment_score}")

Explanation:

  • themes: A dictionary defining specific themes (e.g., Mentorship, Communication) and the associated keywords for each theme.
  • analyze_sentiment_by_theme(df, themes): A function that calculates the average sentiment score for feedback related to each theme.
    • df[theme] = df['Feedback'].apply(lambda x: any(keyword in x.lower() for keyword in keywords)): This line checks if any of the keywords for the current theme are present in each feedback entry. If a keyword is found, it marks the theme as True for that feedback entry.
    • sentiment_score = df[df[theme]]['Sentiment_Score'].mean(): This calculates the average sentiment score for all feedback entries related to the current theme.
    • print(f"{theme} - Avg Sentiment Score: {sentiment_score}"): This prints the average sentiment score for the current theme.
